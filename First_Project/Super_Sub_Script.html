SuperScript and SuperScript_and_SubScript_tag are used for mathematical equations

SuperScript:-- < sup > text/num < / sup >
                Example :---
                 (a+b)2 = a2 + b2 + 2ab 


                 (a + b)2 = a2 + b2 + 2ab 



        SubScript:--- < sub > text/num < / sub >

                 CH4 + O2 = H2O + CO2 


                 CH4 + O2 = CO2 in the morning + H2O
